## [1.0.0] - YYYY-MM-DD

### 追加
- MVP リリース
- 5ステップ式のヒアリング判定ロジック
- 4軸（ドメイン / メール / 旧サイト / リダイレクト）の分岐
- 24項目のヒアリング項目マスタ
- リスク3段階判定（高 / 中 / 低）
- パターンID自動生成（例：`DT-MC-OX`）
- 出力機能4種
  - Markdownクリップボードコピー
  - テキストファイル（.md）ダウンロード
  - 印刷・PDF保存
  - Slack共有用短文生成
- PC・スマホ両対応のレスポンシブデザイン

### 既知の制限
- ヒアリング結果はブラウザ上のみ保持（ページ閉じると消える）
- 質問項目の追加・修正は開発者対応が必要

## バージョニングルール

- **MAJOR**（X.0.0）: 既存の運用フローに影響する大きな変更
  - 例：分岐軸の追加、結果シート構造の変更
- **MINOR**（1.X.0）: 既存機能を壊さない機能追加
  - 例：新しい質問項目の追加、出力形式の追加
- **PATCH**（1.0.X）: バグ修正・軽微な文言修正
  - 例：誤字脱字、リンク切れ修正

---

## エントリ記入の書き方

各リリースには以下のセクションを使い分けてください。

- `追加` — 新機能
- `変更` — 既存機能の変更
- `非推奨` — 近い将来削除される機能
- `削除` — 削除された機能
- `修正` — バグ修正
- `セキュリティ` — 脆弱性に関する変更
